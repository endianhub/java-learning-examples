<br>


#### Wait、Notify 用法

Wait 会让当前线程变为阻塞状态，放弃 CPU 执行权，同时会释放锁。必须在代码块中使用且必须要通过 [对象锁].wait() 。

Notify 唤醒锁池阻塞的线程，从就绪到运行状态

因为涉及到对象锁，wait、notify 必须要在 synchronized 中使用


<br>


多个线程对同一个对象实现不同的操作，过程可以称作为多线之间实现通讯。

<br>


**为什么 Wait 需要释放锁？**

目的就是为了能够让生产者写数据，防止死锁
